## Objetivo do MVP
> Disponibilizar a primeira versão do banco de dados, possibilitando a visualização inicial dos principais indicadores essenciais.

- Coletar dados econômicos públicos;
- Identificar o crescimento do setor farmacêutico em Jacareí, São José dos Campos e Taubaté;
- Organizar dados em banco estruturado;
- Gerar dashboards estratégicos no Power BI.

## Arquitetura do MVP
> Desenvolvimento do projeto e extração de dados no RAIS

- Extração dos dados brutos na plataforma;
- Filtragem do banco de dados para maior especificação dos valores.

 ## Escopo do MVP

 - Coleta de Dados.

## Objetivo

- Capturar indicadores econômicos relevantes para análise.

## Fontes de Dados
- RAIS.

## Indicadores Coletados
- Taxa de empregabilidade por município;
- Taxa de desemprego por município;
- Gênero;
- CBO;

## Resultado Esperado

> Uma plataforma inteligente para:

- Monitorar tendências regionais;
- Evolução mensal;
- Municípios com maior crescimento;
- Empregos gerados;
- Média salarial;
- Novas empresas abertas.
